# 📖 [DECODE A MORSE CODE] <a name="about-project"></a>

**[DECODE A MORSE CODE]** is a tool for decrypting messages in Morse code using Ruby to help yourself by automating this process.
## 🛠 Built With <a name="built-with"></a>

2- Gitflow
3- Rubocop
